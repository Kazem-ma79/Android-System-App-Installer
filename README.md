# Android-System-App-Installer
> Create flashable TWRP package for installing/updating system apps.

## Requirements
#### .Net Framework 4.7.2
#### Java SE Runtime 8+
#### Running Windows OS


## Features
### Free
  > SAIU is a free app
### Easy2Use
  > Easy to use GUI interface
### Material Design
  > Material Design GUI
### MultiApp Support
  > Manage multiple apps in one package
### User Friendly
  > Used simple UX
### Fast Speed
  > Saves your time by fast exporting your package
